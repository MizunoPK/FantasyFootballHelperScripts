NFL Experience Data Integration Objective

OVERVIEW:
Integrate NFL player experience data into the Fantasy Football Helper Scripts system to enhance player scoring and evaluation. This will add a new dimension to draft and trade analysis by factoring in player career stage and experience level.

CORE REQUIREMENTS:

1. DATA COLLECTION (player-data-fetcher):
   - Fetch years of NFL experience for each player using ESPN APIs
   - Add "nfl_experience" column to players.csv output
   - Implement experience data collection with appropriate caching and error handling
   - Ensure data persists across all CSV updates from any module

2. EXPERIENCE CATEGORIZATION:
   - Rookie (0-1 years): Players in their first or second NFL season
   - Seasoned (2-5 years): Players with moderate NFL experience
   - Veteran (6+ years): Experienced players with extensive NFL careers

3. SCORING INTEGRATION (draft_helper/trade_helper):
   - Create experience-based multiplier system similar to existing ADP and Player Ranking approaches
   - Apply multipliers during player scoring for draft recommendations and trade analysis
   - Configure separate multiplier values for each experience bracket
   - Integrate with existing enhanced scoring calculator framework

4. CONFIGURATION AND SIMULATION:
   - Add experience multiplier settings to draft_helper configuration
   - Enable manipulation of the 3 experience multipliers in simulation system
   - Provide same level of configurability as current ADP multiplier controls
   - Allow testing different experience weighting strategies

5. DATA EXPORT AND PERSISTENCE:
   - Update all export functions (CSV, Excel, JSON) to include nfl_experience column
   - Ensure consistent column ordering and data preservation
   - Maintain backward compatibility with existing data files
   - Handle missing experience data gracefully with appropriate defaults

TECHNICAL APPROACH:

ESPN API Integration:
- Use ESPN Team Roster API for efficient batch processing (32 calls vs 600+ individual)
- Implement player ID mapping between fantasy and NFL athlete IDs
- Add caching layer to minimize API calls and improve performance
- Provide graceful fallbacks for missing or failed data lookups

Enhanced Scoring Integration:
- Extend EnhancedScoringCalculator with experience-based adjustments
- Add experience multiplier calculation similar to existing ADP/rating systems
- Integrate with existing draft_helper scoring pipeline
- Maintain scoring transparency and debugging capabilities

Configuration Management:
- Add EXPERIENCE_MULTIPLIERS configuration with Rookie/Seasoned/Veteran values
- Implement experience multiplier toggle (ENABLE_EXPERIENCE_MULTIPLIER)
- Extend simulation system to manipulate experience multipliers
- Provide sensible default values based on fantasy football best practices

Data Model Updates:
- Extend FantasyPlayer class with nfl_experience field
- Update CSV fieldnames in all export functions
- Ensure proper type conversion and validation for experience data
- Handle edge cases (undrafted players, missing data, etc.)

- More nuanced draft strategy recommendations
- Enhanced trade analysis considering player career stage
- Improved long-term roster construction guidance

System Integration:

- Backward compatibility with existing data and workflows
- Extensible framework for future player attribute enhancements
